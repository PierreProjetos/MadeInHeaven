Javascript

O Javascript é uma das linguagens de programação mais populares do mundo.

Mas porque estudar o Javascript?

O Javascript é uma das 3 linguagens de programação que todo desenvolvedor de web deve aprender. Sendo elas:

  1. HTML para definir o conteúdo das páginas da web
  2. Enquanto o CSS é usado para especificar o layout das páginas da web
  3. E o Javascript é usado para programar o comportamento das páginas da web

Comparação entre as linguagens HTML, CSS e JAVASCRIPT

HTML CSS JAVASCRIPT
É uma linguagem de marcação Linguagem de estilização Linguagem de programação que traz comportamento para a página
Aqui você marca cada conteúdo da pagina como parágrafos titulos, tabalas, formulários entre outros Neste você coloca uma cor ou posicionamento ao conteudo Permite interação com o usuário
